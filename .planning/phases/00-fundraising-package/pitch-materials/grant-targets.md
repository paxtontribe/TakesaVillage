## Immediate Action Items

These should be completed before any grant applications are submitted:

### 1. Register on Grants.gov (required for all federal grants)
- **URL:** https://www.grants.gov/register
- **Why:** Cannot apply for any SAMHSA or federal grant without Grants.gov registration
- **Time:** 1-2 weeks for registration to process
- **Action:** Register the nonprofit entity (requires EIN/UEI)

### 2. Set up SAMHSA NOFO email alerts
- **URL:** https://www.samhsa.gov/grants/grants-dashboard/forecasts
- **Why:** NOFOs have 30-60 day windows; missing an announcement means waiting 6-12 months for the next cycle
- **Action:** Subscribe to email alerts for Behavioral Health Equity and Technology Innovation categories

### 3. Join OriginTrail Discord
- **URL:** https://discord.com/invite/origintrail (or search "OriginTrail Discord")
- **Why:** Direct relationship with Trace Labs is the path to ecosystem partnership/funding
- **Action:** Introduce TaV project in the ecosystem channel; request conversation about partnership

### 4. Register on Polkassembly
- **URL:** https://polkadot.polkassembly.io
- **Why:** Required to submit Polkadot Treasury proposals
- **Action:** Create account, review recent successful proposals for format and tone

### 5. Bookmark RWJF active opportunities page
- **URL:** https://www.rwjf.org/en/grants/active-funding-opportunities.html
- **Why:** RWJF's "Local Data for Equitable Communities" program recurs; need to catch the next cycle early
- **Action:** Check monthly for new opportunities matching TaV's data infrastructure angle

### 6. Complete nonprofit incorporation
- **Why:** Required for Grants.gov registration, most foundation applications, and Polkadot Treasury credibility
- **Action:** File articles of incorporation, obtain EIN, apply for 501(c)(3) status

## Grant Positioning Notes

### For healthcare funders (SAMHSA, RWJF, Pew)
- Lead with the problem: $42B industry, no verified reviews, predatory practices documented by FTC
- Frame as "data infrastructure and transparency initiative" -- not "blockchain project"
- Emphasize compliance awareness: three legal gates mapped pre-funding
- De-emphasize: token economics, DKG technical details, Web3 terminology

### For tech-for-good funders (Omidyar, Mozilla, Schmidt)
- Lead with the solution: decentralized, non-commercial, community-accountable
- Frame as "trustworthy AI + decentralized data applied to healthcare"
- Emphasize: no pay-to-rank model, AI agents citing verified data, nonprofit structure
- Highlight: the research corpus as evidence of technical depth

### For Web3 ecosystem funders (Polkadot Treasury, OriginTrail)
- Lead with ecosystem value: real-world social good application using Polkadot PoP, OriginTrail DKG, NeuroWeb
- Frame as: "demonstrating Polkadot identity infrastructure serving people in crisis"
- Emphasize: PoP as core identity layer, DKG as data layer, NeuroWeb for AI inference
- Highlight: this is not a token project or DeFi protocol -- it is healthcare infrastructure on Polkadot
