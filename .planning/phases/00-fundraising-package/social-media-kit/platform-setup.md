# Social Media Platform Setup Guide

**Project:** Takes a Village
**Handle to reserve:** `takesavillage` on all platforms
**Fallback handles:** `takesavillagehealth` or `takesavillage_` if primary is taken

---

## Instagram

### Account Creation

1. Go to https://www.instagram.com/accounts/emailsignup/
2. Sign up with the project email address
3. Choose username: `takesavillage`
4. Complete the signup flow
5. Go to Settings > Account > Switch to Professional Account
6. Select category: **Nonprofit Organization**
7. Choose **Business** (not Creator) for access to analytics and contact buttons

### Profile Setup

- **Profile picture:** TaV logo (teal #2D7D6F background with white text) -- use the logo from the website or create a square version in Canva (400x400px minimum)
- **Header/banner:** Not applicable for Instagram (no banner image); use Highlights covers instead
- **Bio:** Copy from bio-templates.md (Instagram section)
- **Website link:** https://takesavillage.health
- **Action button:** Set "Email" action button to project email
- **Highlights:** Create at least one Highlight called "Help Now" with the SAMHSA helpline number (1-800-662-4357)

### Content Settings

- Enable "Hide Likes" on posts (reduces comparison pressure in a sensitive topic area)
- Set comments to "Everyone" (community engagement is important)
- Enable "Hidden Words" filter and add terms that could attract spam or harmful content

## TikTok

### Account Creation

1. Go to https://www.tiktok.com/signup
2. Sign up with the project email address
3. Choose username: `takesavillage`
4. Complete the signup flow
5. Go to Settings > Manage Account > Switch to Business Account
6. Select category: **Nonprofit & Religion** or **Education** (whichever is available)

### Profile Setup

- **Profile picture:** TaV logo (teal #2D7D6F background with white text) -- square format, 200x200px minimum
- **Bio:** Copy from bio-templates.md (TikTok section)
- **Website link:** https://takesavillage.health

### Content Settings

- **Age gate:** Set content classification to 18+ for substance-adjacent content. Go to Creator Tools > Content Disclosure > enable age restriction on all videos discussing substance use
- Pin the SAMHSA helpline (1-800-662-4357) in your first video or add it to your bio
- Enable comment filtering for harmful or triggering language

### Important Notes

- TikTok restricts substance-adjacent content to users 18+. All videos discussing substance abuse treatment must have the age restriction enabled
- Avoid showing or naming specific drugs -- focus on recovery, treatment access, and transparency
- Use the "For You" discovery strategy: trending sounds + educational hooks + clear captions

## X (Twitter)

### Account Creation

1. Go to https://x.com/i/flow/signup
2. Sign up with the project email address
3. Choose username: `takesavillage`
4. Complete the signup flow (including phone verification)

### Profile Setup

- **Profile picture:** TaV logo (teal #2D7D6F background with white text) -- 400x400px recommended
- **Header/banner:** Mission statement tagline on teal brand background (1500x500px). Suggested text: "Transparent treatment information for everyone. No pay-to-rank. No hidden agendas." Create in Canva or use the website og-image.png as a placeholder
- **Bio:** Copy from bio-templates.md (X section)
- **Website link:** https://takesavillage.health
- **Location:** Set to "United States" (US-first launch)

### Verified Organizations (Free for Nonprofits)

1. Go to https://x.com/i/verified-orgs-signup
2. Apply under the **Nonprofit** category
3. X offers free Verified Organizations badges for registered nonprofits
4. This provides a gold checkmark badge and enhanced credibility
5. Note: Approval may take several weeks; apply early

### Content Settings

- Enable "Quality filter" for replies
- Pin the announcement thread (x-thread-01.md) once published

## LinkedIn

### Company Page Creation

1. Go to https://www.linkedin.com/company/setup/new/
2. Select page type: **Nonprofit**
3. Enter company name: "Takes a Village"
4. Enter URL name: `takesavillage` (this creates linkedin.com/company/takesavillage)
5. Complete the required fields

### Profile Setup

- **Logo:** TaV logo (teal #2D7D6F background with white text) -- 300x300px recommended
- **Banner image:** Mission statement tagline on teal brand background (1128x191px). Same design as X banner, adjusted for LinkedIn dimensions. Create in Canva
- **Tagline (120 char):** "Nonprofit platform for verified substance abuse treatment transparency. Built on blockchain."
- **Description:** Copy from bio-templates.md (LinkedIn section)
- **Website:** https://takesavillage.health
- **Industry:** Nonprofit Organization Management
- **Company size:** 1 (solo founder stage)
- **Specialties:** Substance Abuse Treatment, Healthcare Transparency, Blockchain, Decentralized Knowledge Graph, Nonprofit Technology

### Additional Settings

- Add the GitHub repo link (https://github.com/paxtontribe/TakesaVillage) in the "About" section
- Enable "Share admin updates" to amplify posts through personal networks
- Invite personal connections to follow the page after the first post is published

- [ ] Username is `takesavillage` (or documented alternative) on all four platforms
- [ ] Profile picture is the same TaV logo on all four platforms
- [ ] Bio links to https://takesavillage.health on all four platforms
- [ ] SAMHSA helpline (1-800-662-4357) is visible in bio or pinned content on all platforms
- [ ] Banner/header images use consistent brand colors (teal #2D7D6F, amber #F4A261)
- [ ] All accounts are set to the correct account type (Professional/Business/Nonprofit)
- [ ] Content settings are configured per platform guidelines above
